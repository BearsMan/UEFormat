Unreal Export Format
------------------------------------------

A model and animation export format created for Unreal Engine game asset extraction.

## Exports
### Model (.uemodel)
- Vertices
- Indices
- Normals
- Tangents
- Texture Coordinates
- Materials
- Weights
- Bones
- Sockets
- Morph Targets

UE Export Format
------------------------------------------

A model and animation export format created for Unreal Engine game asset extraction. The export format is currently supported by [CUE4Parse](https://github.com/FabianFG/CUE4Parse).

### Model Data (.uemodel)
- Vertices
- Indices
- Normals
- Tangents
- Texture Coordinates
- Vertex Colors
- Materials
- Weights
- Bones
- Sockets
- Morph Targets

### Animation Data (.ueanim)
- Location Keys
- Rotation Keys
- Scale Keys
- Float Curve Keys
